## 📖 简介

**翼麒麟（Wing-Kirin）** 是「[龙之生存](https://github.com/DragonSurvivalTeam/DragonSurvival)」的一个附属模组。本仓库存放着该模组的 **全部源码**和**资源文件** ，旨在为有需要的人提供一份 **免费、开放** 的学习资料。

本模组为原版龙之生存增加了全新的种族、能力、以及与之配套的视觉效果与玩法机制。项目代码基于 **NeoForge 1.21.1**，主要编程语言为 **Java** 与 **mcfunction**。

原来此仓库是私有仓库，但现在翼麒麟已**不再更新**。因此我决定将所有文件公开，为后来者提供一个**可行的框架**。

这个模组是作者的第一个Minecraft模组，因此每个文件都有详尽的注释，方便理解逻辑。此后不会再添加新内容，只会修复bug。

无论你是一位普通的玩家，还是一名正在学习 Minecraft 模组开发的开发者，都欢迎你来到这里。
