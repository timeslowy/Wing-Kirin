## 药水效果方面

### 定身（Ding Shen）

> 重头戏来了！  
  定身术迄今为止仍是翼麒麟里技能逻辑**最复杂**的技能，同时也算龙种的**招牌技能**。  
  因此，此次定身术的药水效果需要做**比较重大、有些难度**的更改，以简化数据包的运行逻辑，同时避免一些不必要的BUG。

#### 药水效果初应用时（只执行一次）

- 播放声音（被定身音效）：**烈焰人受伤声音**，音量：2.0，音调：1.2  

        # 原来由数据包指令播放
        playsound entity.blaze.hurt player @a ~ ~ ~ 2.0 1.2

- 将实体的`发光描边轮廓颜色`改为**金色**（#fbdc92）  

  因为之前的发光效果颜色更改效果通过指令使实体**加入队伍**来实现，可能会破坏队伍的效果，导致玩家被从先前的队伍移除等后果，  
  因此需要通过Java来修改。  
  **当然，此效果需要在药水效果结束时重置。**  

- 被定身实体发出粒子效果
  dust粒子，颜色(color)：RGB颜色[0.98,0.86,0.57]（十六进制：#fbdc92）,倍率（scale）： 1.0 ，delta：1 1 1 ，速度（speed）：0，数量（count）：200

        # 原来用数据包指令发出
        particle dust{color:[0.98,0.86,0.57],scale:1.0} ~ ~ ~ 1 1 1 0 200

- 将实体的NBT标签`NoAI`设置为`true`  
  *从指令转出结合在一起总归是好事*


#### 药水效果存在时（持续执行）

- 使得玩家**完全无法通过方向键移动**  

  原先我在MCR流程`DINGSHENMOVEMENTSTOP`里使用了代码块

        # 尝试将事件/目标实体的的速度矢量覆写为vx:0 vy:0 vz:0
        ...
        entity.setDeltaMovement(new Vec3(0, 0, 0));
        ...
    
  对生物很有效果，但对玩家效果**稍有欠缺**，实测在药水效果存在下玩家仍然能在地面和空中进行**极为缓慢的移动**。  
  希望能做到玩家**完全无法进行移动**的效果，例如龙生技能定义文件里有`在施法时是否可移动`的选项，做到类似施法时完全不能移动的效果。

#### 药水效果结束时（包括效果结束时和死亡时，只执行一次）

- 被定身实体发出粒子效果
  dust粒子，颜色(color)：RGB颜色[0.98,0.86,0.57]（十六进制：#fbdc92）,倍率（scale）： 1.0 ，delta：1 1 1 ，速度（speed）：0，数量（count）：200

- 将实体的NBT标签`NoAI`设置为`false`
